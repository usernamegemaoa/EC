#桩和驱动

## 测试
0. service\impl下新建class类，使用自己的接口
1. 模仿我写的file测试类

## 驱动
0. 右键service，`new >> others >> java >> junit >> junit test case`
1. `New Junit 4 test` 看清楚是4
2. `source folder`改为`ec/src/test/java`
3. NEXT
4. 勾选所有接口
5. FINISH
6. 写测试代码，模仿我的
