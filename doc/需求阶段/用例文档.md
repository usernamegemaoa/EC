## 用例详情

### 首页商品展示
- 顾客
- 正常流程
    1. 用户打开首页
    2. 网页显示首页，首页包括最新单品和热销单品，商品信息包括图片、商品名和价钱
    3. 用户选择每日上新
    4. 网页显示每日上新，每日上新包括新添加的商品，商品信息包括图片、商品名和价钱
    5. 用户选择上装（T恤、针织衫、外套、内衣），裙装，下装等
    6. 系统显示对应商品类别信息和特点商品图片广告
- 扩展流程
    1. 用户选择玩具类别
        1. 系统提示无法显示，返回主页
    2. 用户选择食品
        1. 系统提示不行
        2. 用户还要选
        3. 系统返回主页

### 商品修改                
- 商家  
- 正常流程
    1. 用户点击商品修改
    2. 网页显示用户添加过的所有商品（包括商品添加时间，商品名称，商品价格，商品图片，商品分类）
    3. 用户选择一个商品进行修改
    4. 网页显示该商品的修改界面（可以修改商品名称，商品价格，商品图片，商品分类）
    5. 用户点击确认修改
    6. 网页显示修改结果
    
### 我的订单
- 商家
- 正常流程
    1. 用户点击我的订单
    2. 网页显示该商家所有的出货订单（包括订单编号，商品图片，商品名称，商品总价）
    3. 用户点击一个订单
    4. 网页显示该订单的所有信息（包括订单编号，订单时间，买家名称，商品图片，商品名称，商品单价，商品类型，商品数量，商品总价）

### 发货
- 商家
- 正常流程
    1. 商家点击发货
    2. 网页显示该商家所有未确认发货的订单
    3. 用户选择一个订单
    4. 网页显示该订单的发货界面(包括订单编号，买家名称，收货地址，商品图片，商品名称，商品单价，商品数量，商品总价，快递单号输入框）
    5. 用户输入快递单号并确认
    6. 网页显示确认发货成功
- 扩展流程
    6. 输入的快递单号有错误
        1. 弹出提示框（快递单号错误）

### 完成退货
- 商家

    3. 用户选择一个退货申请点击完成

### 收取货款（用户确认收货）
- 顾客
- 正常流程
    1. 用户进入我的订单界面
    2. 用户点击确认收货
    3. 网页显示该商品的确认收货界面（包括订单编号，商品名称，商品数量，商品单价，商品总价，成交时间）
    4. 用户点击确认
    5. 网页显示确认收货是否成功

### 个人信息
- 商家
- 正常流程
    1. 用户点击个人信息
    2. 网页显示该商家信息（包括商铺名称，收款账户，我的订单，修改密码）

### 修改密码
- 商家
- 正常流程
    1. 用户点击修改密码
    2. 网页进入修改密码界面，并要求输入当前密码，新密码，再次输入新密码
    3. 用户输入当前密码，新密码后确认
    4. 网页显示修改成功
- 扩展流程
    3. a.用户输入当前密码错误
        1. 网页显示当前密码输入错误
    3. b.用户输入的两次新密码不相同
        1. 网页显示两次新密码不相同
    3. c.用户输入的新密码不符合规则
        1. 网页显示密码规则并提示用户输入的密码不符合要求
        

### 回复评价
- 商家
- 正常流程
    1. 用户点击查看评价
    2. 网页显示该店铺中的所有评价（包括用户名称(可匿名），商品名称，评价时间，评价内容）
    3. 用户点击回复
    4. 网页中显示回复的输入框
    5. 用户点击提交
    6. 界面刷新并显示回复

### 回复咨询
- 商家
- 正常流程
    1. 用户点击查看咨询
    2. 网页显示所有咨询（包括咨询内容，咨询时间，用户名称）
    3. 用户点击回复
    4. 网页中显示回复的输入框
    5. 用户点击提交
    6. 网页中显示回复是否成功

### 添加广告
- 商家
- 正常流程
    1. 用户点击添加广告
    2. 网页显示广告添加界面（包括广告持续天数，广告图片）
    3. 用户输入天数并上传个广告图片
    4. 网页中显示广告总价
    5. 用户确认支付
    6. 跳转到支付界面

### 删除广告
- 商家
- 正常流程
    1. 用户进入我的广告
    2. 网页显示所有本店铺正在进行的广告
    3. 用户选择一个广告进行删除
    4. 网页中显示退款数额以及是否确认删除
    5. 用户选择删除
    6. 网页中显示删除成功
